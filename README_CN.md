# GMESDataset: 地质多物理场地球模拟数据集

[English Version](./README.md)

**GMESDataset** 是一个用于构建合成地质模型并执行多物理场（重力、磁法、电法、地震）联合正演的综合框架。该项目旨在生成包含各种复杂地质异常和相态的逼真三维地球模型，服务于联合反演、多模态学习和地球物理综合解释的基准测试。

## 🌟 核心特性

### 1. 高级地质建模 (`core/`)
基于过程生成的建模引擎，能够创建具有多物理场一致性的复杂地质结构：
*   **岩浆侵入体**：三维岩墙群 (Dyke Swarms)、岩床 (Sills) 以及岩株/岩栓 (Stocks)，包含热接触变质晕。
*   **油气系统**：气藏（透镜体/圈闭）、气烟囱 (Gas Chimneys) 和天然气水合物（BSRs、斑状饱和度）。
*   **断裂系统**：含卤水/流体的断层，精细刻画断层核 (Core) 和破碎带 (Damage Zone)。
*   **矿化体**：块状硫化物矿床（高密度/高导电性）。
*   **基底/构造**：沉积-基底界面模拟及盐丘构造。
*   **蚀变带**：与深部断裂耦合的蛇纹岩化带（高磁化率、低密度、低速）。

### 2. 多物理场属性生成
自动生成空间一致的三维属性体：
*   **地震**：纵波速度 ($V_p$)
*   **重力**：密度 ($\rho$)
*   **磁法**：磁化率 ($\chi$)
*   **电法**：电阻率 ($\Omega\cdot m$)

### 3. 正演模拟引擎
*   **地震**：基于 `deepwave` (PyTorch) 的高性能 2D/3D 波场传播模拟。
*   **重力**：三维重力异常正演。
*   **磁法**：三维磁异常模拟 (MATLAB/Python)。

## 📂 目录结构

```plaintext
GMESDataset/

│   └── petrophysics/           # 岩石物理关系定义
├── configs/                    # 异常体配置文件

├── Magnetic/                   # 磁场模拟

```

## 🚀 快速开始

### 1. 环境依赖
项目依赖标准的科学计算 Python 栈。主要依赖项包括：
*   `numpy`, `scipy`, `matplotlib`
*   `segyio` (用于读取 SEGY 背景模型)
*   `cigvis` (推荐用于 3D 可视化)
*   `torch`, `deepwave` (如果需要运行地震模拟)

### 2. 生成地球模型
运行演示脚本以创建一个包含多种异常体的合成地球模型。

```bash
python run_demo.py
```

该脚本将执行以下操作：
1.  加载背景模型（如果没有 SEGY 文件则生成合成背景）。
2.  注入定义的地质异常（岩墙、断层、气藏等）。
3.  计算多物理场属性（Vp, Rho, Chi, Res）。
4.  交互式可视化结果（如果安装了 `cigvis`）。

### 3. 运行地震模拟
进入地震建模目录，对生成的模型进行波场模拟。

```bash
cd Seismic/forward_modeling
# 编辑 config.yaml 以指向您生成的模型文件
python main.py --config config.yaml
```

## 📊 地球物理属性特征表

下表总结了当前实现的地质异常体的典型物理属性变化，基于代码默认参数及标准岩石物理学设定。

| 异常类型 (Anomaly Type) | 速度 ($V_p$) | 密度 ($\rho$) | 电阻率 ($\Omega\cdot m$) | 磁化率 ($\chi$) |
| :--- | :--- | :--- | :--- | :--- |
| **岩浆侵入体 (Igneous)** | **高** (~5500 m/s) <br> *接触晕: +3%* | **高** | **高** | **高** <br> *(若含磁性矿物)* |
| **块状硫化物 (Sulfide)** | **极高** (~6200 m/s) | **高** | **极低** (高导) | **高** <br> *(若富含磁黄铁矿)* |
| **气藏 (Gas)** | **低** (气 ~1800 m/s) | **低** | **高** | **低** (抗磁性) |
| **天然气水合物 (Hydrate)** | **高** (水合物 ~3700 m/s) <br> *下伏游离气: 低* | **低** | **高** | **低** |
| **含卤水断层 (Brine Fault)** | **低/中性** <br> (若破碎: 0% 到 -4%) | **中性/低** | **极低** (高导) <br> *(核部 ~0.5 $\Omega\cdot m$)* | **中性** |
| **蛇纹岩化带 (Serpentinized)** | **低** (-25%) | **低** (-12%) | **可变** | **高** (+0.02 SI) <br> *(生成磁铁矿)* |
| **盐丘 (Salt Dome)** | **高** (~4500-5500 m/s) | **低** (~2100-2200 kg/m^3) | **极高** | **低** (抗磁性) |
| **沉积围岩 (Sediment)** | 背景趋势 | 背景趋势 | 背景趋势 | 背景趋势 |

> **说明**：具体物理属性可通过配置文件或直接修改 `core/anomalies/*.py` 进行自定义。参考物理逻辑：[GPT-4 对话记录](https://chatgpt.com/s/t_6988301699188191aa2a68b0f53dbecf)。
